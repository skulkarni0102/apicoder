API Coder Privacy Policy

Last Updated: December 2024
Extension Version: 1.0

Introduction

This Privacy Policy describes how API Coder ("we," "our," or "us") collects, uses, and protects your information when you use our Chrome extension. We are committed to protecting your privacy and being transparent about our data practices.

Information We Collect

API Coder collects the following types of information when you use the extension:

1. Authentication Information

  • Authentication cookies and tokens from API requests
  • Session IDs and bearer tokens
  • Other authentication credentials present in HTTP requests

2. User Activity

  • HTTP requests and responses when capture mode is enabled
  • API call patterns and network activity
  • Timestamps of captured requests

3. Website Content

  • Request headers and bodies from API calls
  • Response data and status codes
  • URLs of API endpoints accessed

How We Use Your Information

We use the collected information solely for the following purposes:

  • Code Generation: To generate working code snippets in various programming languages
  • Security Testing: To create security test cases for API endpoints
  • Extension Functionality: To provide the core features of API Coder

Data Storage and Security

Local Storage: All captured data is stored locally on your device using Chrome's secure storage API. We do not maintain any servers or databases to store your information.

Data Retention

  • Captured API requests are stored locally until you clear them
  • Your Gemini API key is stored securely for code generation
  • User preferences (language selection, debug mode) are saved locally

Data Sharing and Third Parties

We do not sell, trade, or otherwise transfer your personal information to third parties, except:

  • Google Gemini API: When you request code generation, we send the captured API request data to Google's Gemini AI service to generate code snippets. This is the only external data transmission.
  • Legal Requirements: We may disclose information if required by law or to protect our rights and safety.

Your Control Over Your Data

You have complete control over your data:

  • Start/Stop Capture: You control when API capture is active
  • Clear Data: You can clear all captured requests at any time
  • Debug Mode: You can inspect exactly what data is being captured
  • Uninstall: Removing the extension deletes all stored data

Children's Privacy

API Coder is not intended for use by children under 13. We do not knowingly collect personal information from children under 13.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by updating the "Last Updated" date at the top of this policy.
